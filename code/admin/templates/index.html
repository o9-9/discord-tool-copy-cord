Proxy Rotation

When enabled, every Discord API call during structure sync (channel create, webhook create, guild edit, etc.) is routed through a rotating proxy and rate limiting is bypassed — syncs complete much faster.
Paste one proxy per line. Examples: host:port, host:port:user:pass, user:pass@host:port
